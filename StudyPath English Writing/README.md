# 英语作文模板句子背诵应用 (Vue + Element Plus 版本)

这是一个基于 Vue 3 和 Element Plus 构建的英语作文模板句子背诵应用，帮助用户通过输入英文句子来背诵和练习英语作文模板。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Element Plus** - 基于 Vue 3 的组件库
- **Vite** - 下一代前端构建工具
- **Composition API** - Vue 3 组合式 API

## 使用指南

### 1. 启动应用

在终端中运行：

```bash
npm run dev
```

### 2. 选择模板

- **使用内置模板**：点击"使用内置模板"按钮
- **上传自定义文件**：拖拽 .txt 文件到上传区域，或点击上传按钮

### 3. 开始练习

1. 阅读中文句子
2. 在输入框中输入对应的英文句子
3. 点击"完成"按钮或按 `Ctrl + Enter` 检查答案
4. 查看比对结果和统计数据

### 4. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Enter` 或 `Ctrl + E` | 检查答案（在输入框聚焦时） |
| `Ctrl + →` 或 `Ctrl + N` | 下一个句子（输入框未聚焦时） |
| `Ctrl + ←` 或 `Ctrl + P` | 上一个句子（输入框未聚焦时） |
| `Ctrl + R` | 重置输入（输入框未聚焦时） |
| `Tab` | 插入 4 个空格（在输入框中） |

### 5. 导航控制

使用底部控制按钮：
- **上一个**：查看上一个句子
- **清空输入**：清除当前输入
- **完成**：检查答案并显示比对结果
- **下一个**：查看下一个句子
- **重新上传**：重新选择模板文件

## 文件格式要求

如果要上传自定义模板文件，需要遵循以下格式：

```
1，中文句子
英文句子

2，中文句子
英文句子

3，中文句子
英文句子
```

**格式说明：**
- 每个句子对以数字开头，后面使用中文逗号或句号
- 中文句子和英文句子之间用空行分隔
- 英文句子可以包含换行
- 文件编码必须是 UTF-8

## 开发说明

### 核心组件

**App.vue** - 主应用组件，包含所有业务逻辑：
- 句子解析和显示
- 文件上传处理
- 答案比对算法
- 统计数据计算
- 键盘事件处理

### 主要函数

- `loadBuiltinTemplate()` - 加载内置模板
- `handleFileChange()` - 处理文件上传
- `loadContent()` - 加载并解析内容
- `parseSentences()` - 解析句子文本
- `checkAnswer()` - 检查答案
- `compareTextsWithAnswer()` - 比对文本并生成高亮
- `updateStats()` - 更新统计信息
- `nextSentence()` / `previousSentence()` - 切换句子
- `resetInput()` - 重置输入
- `reuploadFile()` - 重新上传

### 样式定制

应用使用了 Element Plus 的主题色，并添加了自定义样式：
- 渐变背景：`#667eea` 到 `#764ba2`
- 自定义滚动条样式
- 响应式布局支持

## 常见问题

**Q: 为什么上传的文件无法解析？**
A: 请确保文件格式正确，使用 UTF-8 编码，并遵循"数字，中文句子"和"英文句子"的格式。

**Q: 如何添加更多的句子？**
A: 编辑 `public/template.txt` 文件，按照现有格式添加新的句子对。

**Q: 统计数据不准确怎么办？**
A: 确保输入的内容与标准答案的标点符号、大小写一致。

**Q: 可以在手机上使用吗？**
A: 可以，应用采用响应式设计，支持移动端浏览器。

**Q: 如何自定义主题色？**
A: 可以在 `App.vue` 的 `<style>` 部分修改 CSS 变量和颜色值。
